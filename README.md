CompassQuotes or Quotes+ is an iOS app that shows a new inspirational quote every day, 
This version originated from a rebuild of Fortu (https://apps.apple.com/it/app/fortu-daily-fortune/id6753077218?l=en-GB), with a big focus on learning design, animations, structure, and AI implementations, through FoundationModels in my app.
I added some features and details, like a FavoritesView, with the scope to make the AI work according to user's preferences, and touch-reactive elements.
